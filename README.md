# Proyecto_Final1
Proyecto final del curso de html de devf
Este es un blog personal.
Tiene una interface princilal donde da una breve descripcion de lo que soy.
Hay un boton para cambiar el modo de pagina a oscuro o claro.
Se encuentra botons redirigidos a mis redes sociales.
Sn la visualizacion se aprecia un boton para ver mas acerca de mi donde redirigira a una pagina para ver informacion acerca de mi y como contactarme.
Hay un boton con el nombre de galeria donde redirigira a una pagina con un formulario para alegir que imagenes ver.
En la parte izquierda superior al hacer click en mi nombre nos mandara al inicio.
